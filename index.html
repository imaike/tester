Ken-Q Analysis

Beta Version 0.6.0 Test Release (October 3, 2016)

  • Ken-Q Analysis requires an up-to-date browser - Tested on:
    • Windows 10 --> Chrome (version 50), Firefox (version 46)
    • Mac OSX El Capitan --> Chrome (version 50), Firefox (version 46)
    • Linux (Lubuntu 14.04) --> Chromium (version 50), Firefox (version 46)
    older Chrome and Firefox versions are not tested and not recommended. Principal Component Analysis takes substantially more time to calculate on Firefox (about 2 minutes for large datasets) compared to Chrome (about 10 seconds). Firefox will give "unresponsive script" error messages while calculating Principal Components. Click "continue" to finish the calculations.
  • CAUTION - Ken-Q Analysis saves data locally inside the browser. If you are using a shared computer in a computer lab or internet cafe you should delete all locally-stored data after you download the results of your analysis
  • Ken-Q Analysis is open source - you can download and view the source files.
  • Functionality not yet implemented:
    • Alternative (Horst) centroid factor extraction
    • Analysis of Free-distribution Q-sorts
    • Factor loading adjustment for Heywood case (communality equal to or greater than 1.0)
  • Worried about data privacy? Want to work offline? Download Ken-Q Analysis and run it locally from your computer. (after downloading the file, unzip it and double-click on the "index.html" file to start the web app.)

  • Changed output file format to include Analysis Overview
  • Added display language selection for user interface and analysis output

  • added Principal Components to factor extraction options
  • redesigned user interface and analysis flow to accommodate Principal Components option
  • changed rotation history to project history
  • changed highlighting in Judgemental Rotation plot and chart to match autoflagging criteria

  • added flagged Q-sorts weights tables to Preliminary Output and Complete Results
