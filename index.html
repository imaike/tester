Ken-Q Analysis

A Web Application for Q Methodology

Version 0.5.0 (June 4, 2016)

  • Requires a recent version of Chrome, Firefox, Internet Explorer, or the Microsoft Edge web browser.
  • CAUTION - Ken-Q Analysis saves data locally inside the browser. If you are using a shared computer in a computer lab or internet cafe you should delete all locally-stored data after you download the results of your analysis
  • Ken-Q Analysis is open source - you can view the source files HERE.
  • Functionality not yet implemented:
    • Principal components factor analysis
    • Alternative (Horst) centroid factor extraction
    • Manual input of unrestrained sorts
    • Factor loading adjustment for Heywood case (communality equal to or greater than 1.0)
  • Worried about data privacy? Want to work offline? Download Ken-Q Analysis HERE and run it locally from your computer. (after downloading the file, unzip it and double-click on the "index.html" file to start the web app.)
Change Log
Beta Version 0.5.0 Test Release (June 4, 2016)
  • added Principal Components to factor extraction options
  • redesigned user interface and analysis flow to accommodate Principal Components option
  • changed rotation history to project history
  • changed highlighting in Judgemental Rotation plot and chart to match autoflagging criteria
Beta Version 0.4.0 Test Release (April 26, 2016)
  • added flagged Q-sorts weights tables to Preliminary Output and Complete Results
  • added flagged Q-sorts correlation tables to Preliminary Output and Complete Results
  • added text to Judgemental Rotation section
Beta Version 0.3.3 Test Release (April 17, 2016)
  • added default sorting option to factor loadings table
  • fixed bug which added two footer rows to factor loadings table
  • added row highlighting option to factor loadings table
  • added highest loading factor column to factor loadings table
Beta Version 0.3.2 Test Release (April 10, 2016)
  • fixed bug when respondent IDs not unique
  • fixed bug when respondent ID contains "."
  • added automatic sorting of factor loading table
  • moved %Exp Var from table to footer
  • improved performance of 2 factor rotation chart

  • removed eigenvalues row from factor loadings table
  • fixed upload error when PQMethod DAT file has special characters
  • fixed D3 scatterplot bug when factor loading greater than 1
  • fixed bug in Excel import function
